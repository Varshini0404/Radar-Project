# 📡 FMCW Radar Range–Doppler Map (MATLAB)

## 🔍 Project Overview

This project simulates an **FMCW Radar system** to generate:

* Range FFT
* Range-Doppler Map (RDM)
* Noise impact analysis using multiple SNR values

## ⚙️ Features

* Noiseless signal analysis
* AWGN noise simulation (30, 20, 10, 5 dB)
* Hamming windowing for clean FFT
* Peak detection for range & velocity
* Proper axis labeling

## 📊 Outputs

* Range FFT plots
* Range-Doppler maps
* Detected target range and velocity

## 🧠 Concepts Used

* FMCW Radar
* FFT (1D & 2D)
* Doppler Effect
* Signal Processing
* Noise Modeling

## 📁 File Structure

* `main.m` → Complete radar simulation

## 🚀 How to Run

1. Open MATLAB / Octave
2. Run:

```matlab
main
```

## 🎯 Target Parameters

* Range: 110 m
* Velocity: -20 m/s
